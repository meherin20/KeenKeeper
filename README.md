# KeenKeeper

KeenKeeper is a friendship management app that helps you stay connected with the people who matter most. Track when you last contacted friends, log interactions, and view your relationship habits in a simple, responsive dashboard.

## Technologies Used

- React.js
- Vite
- React Router DOM
- Tailwind CSS
- Recharts
- React Hot Toast

## Key Features

1. **Friend Dashboard** — Browse all friends in a responsive card grid with status badges, tags, and summary stats.
2. **Interaction Timeline** — Log and view calls, texts, and video chats with filter options by interaction type.
3. **Friendship Analytics** — Visualize your communication habits with a donut chart showing Call, Text, and Video breakdowns.
